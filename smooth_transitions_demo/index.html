Smooth Transitions Demo

This demo's gonna use keyframes and a jquery plugin called smoothstate.js to give your website cool transitions. Credit to css-tricks except they took it one step further by incorporating JS.

Plan:

  1. Make the normal content of the page
  2. Create the keyframes
  3. Attach keyframes to classes with css animation properties
  4. Add classes to html

1. Make the normal content of the page

Chunk your content into divs to make it easier to animate.

2. Create the keyframes

I wanted the heading text to fade in and the content to slide up, so here's what I wrote in my stylesheet.

@keyframes fadeIn { 0% { opacity: 0; } 100% { opacity: 1; } }
@keyframes fadeInUp { 0% { opacity: 0; transform: translate3d(0, 100%, 0); } 100% { opacity: 1; transform: none; } }

3. Attach keyframes to classes with CSS animation properties

Try not to have your transition times too long otherwise your website will be annoying

/** Basic styles for an animated element */
.scene_element { animation-duration: 0.25s; transition-timing-function: ease-in; animation-fill-mode: both; }

/** An element that fades in */
.scene_element--fadein { animation-name: fadeIn; }

/** An element that fades in and slides up */
.scene_element--fadeinup { animation-name: fadeInUp; }

4. Add classes to HTML

I want to make the header fade in so I'll assign it the .scene-element--fadein property. I also want to take the content fade in and float up so I'll assign that the scene_element--fadeinup property.

And that's it! Refresh the page and now your stuff should float in.
