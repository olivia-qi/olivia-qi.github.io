Today, we will mess with some Pseudo Classes and elements

Let's first get a better understanding of CSS

If we think of CSS in terms of selectors, properties, and valuees, then a typical CSS entry would look like this:

            
                selector {property: value;}

To style the above, we need to add two tags together. We call this CSS Combinators.

What are Pseudo Classes?

Again, if we think of CSS in terms of selectors, properties, and values, then a Pseudo Class is an add-on to a selector.

            
                selector:pseudo_class {property:value;}

We use them for links

If we want our link to look different, we can simply add it in the css as follows.

            
                a {color:red}

But if we want each of our link states to look different, then we will use a pseudo class like:

  • :visited
  • :hover
  • :active
